# Contributing

For the canonical contribution guide, see [`CONTRIBUTING.md`](https://github.com/trnsci/trnsci/blob/main/CONTRIBUTING.md) at the repository root.

For the code of conduct, see [`CODE_OF_CONDUCT.md`](https://github.com/trnsci/trnsci/blob/main/CODE_OF_CONDUCT.md).

The same `CONTRIBUTING.md` is mirrored in each of the six sub-projects (`trnfft`, `trnblas`, `trnrand`, `trnsolver`, `trnsparse`, `trntensor`) with only the project title adjusted. The body is identical.

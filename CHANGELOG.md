# Changelog

All notable changes to the trnsci umbrella package.

## [0.1.0] — 2026-04-12

Initial umbrella release.

- Coordinating meta-package for trnfft, trnblas, trnrand, trnsolver, trnsparse, trntensor
- Extras: `[fft]`, `[blas]`, `[rand]`, `[solver]`, `[sparse]`, `[tensor]`, `[all]`, `[neuron]`, `[dev]`
- Cross-project DF-MP2 integration example
- Umbrella mkdocs site linking to per-project docs
- Unified neuronxcc ≥ 2.24 across all sub-projects
